What is this site for?

This site shows you all Guild Tags available for Discord for you to pick.

How to active a Guild Tag?

1) Join one of the servers below.
2) Confirm and accept server requirements.
3) Go to settings -> Profiles -> Guild Tag (at the bottom)
4) Select the one you want from the list you have.
